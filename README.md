# Dart Counter

A dart scoring app built with React + TypeScript + Vite. Tracks scores, stats, XP, levels, titles, and includes sound effects and background music.

## Features

- 501, 301, 701, 101 game modes with double-out / straight-out
- Around the Clock mode
- Practice (free scoring) mode
- Multi-leg matches (best of 1/3/5/7)
- XP and leveling system with 10 built-in titles + custom title creation
- Synthesized sound effects and background music (6 tracks)
- Stats: 3-dart average, first 9, 180s, high checkout, win rate, scoring distribution, dartboard heatmap
- Match history with visit-by-visit detail
- Dark/light themes with accent color picker
- Local backup export
- Full-screen no-scroll play layout with current player on top and others below
